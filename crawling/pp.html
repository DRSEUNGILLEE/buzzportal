원의 색깔은 뉴스의 긍정 점수를 나타내며 원의 크기는 뉴스의 중요도를 의미해요. 원 선택시 자세한 정보를 볼 수 있어요.

사용 AI 모델
  • 중요도 모델: 애널리스트가 투자에 중요하다고 생각하는 뉴스 데이터와 조회수가 높은 뉴스 데이터를 바탕으로 '중요 뉴스'를 판단하도록 학습한 인공지능 모델로 뉴스의 중요도를 계산해요.

수익성 (ROE, %)

자기자본이익률(ROE)이란 무엇인가요?

기업이 보유한 자기자본 대비 1년간 얼마나 수익을 냈는지 의미해요.

ROE = ( 당기 순이익 / 자기자본 ) x 100

성장성 (매출액증가율(YoY), %)

매출액증가율(YoY)이란 무엇인가요?

기업이 1년 전에 비해 얼마나 매출액이 증가했는지 의미해요.

매출액증가율 = ( 올해 매출액 - 작년 매출액 ) / 작년 매출액 x 100

안정성 (유동비율, %)

유동비율이란 무엇인가요?

기업이 단기적인 채무에 대응할 수 있는 유동성 자산이 얼마나 되는지 나타내는 비율이에요.

유동비율 = ( 유동자산 / 비유동자산 ) x 100

활동성 (총자산회전율, %)

총자산회전율이란 무엇인가요?

기업이 가지고 있는 자산이 얼마만큼의 매출액을 내는지 보여주는 비율이에요.

총자산회전율 = ( 매출액 / 총자산 ) x 100
